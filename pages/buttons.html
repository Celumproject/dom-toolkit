Buttons #back to top

Usage

To apply this component, add the .dom-button class and a modifier such as .dom-button-default to an <a> or <button> element. Add the disabled attribute to a <button> element to disable the button.

link           
                                    
<a class="dom-button" href="#">link</a>

<button class="dom-button">button</button>

<button class="dom-button" disabled>disabled</button>

Style modifiers

               
                                    
<button class="dom-button dom-text-primary dom-border-primary">primary</button>
<button class="dom-button dom-text-white dom-bg-primary">primary</button>

<button class="dom-button dom-text-secondary dom-border-secondary">secondary</button>
<button class="dom-button dom-text-white dom-bg-secondary">secondary</button>

<button class="dom-button dom-text-tertiary dom-border-tertiary">tertiary</button>
<button class="dom-button dom-text-white dom-bg-tertiary">tertiary</button>

Login type

     
                                    
<button class="dom-button dom-login-button dom-text-primary dom-border-primary">button</button>
<button class="dom-button dom-login-button dom-bg-primary dom-border-primary">button</button>

Size modifiers

Add the .dom-button-small or .dom-button-large class to a button to make it smaller or larger.

                                    
<button class="dom-button dom-button-small">small button</button>

<button class="dom-button dom-button-large">large button</button>
